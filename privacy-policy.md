# Privacy Policy

**Last updated:** 2026-03-21

## Overview

chatgpt export to word/pdf is a Chrome extension that allows users to export the current ChatGPT conversation to Word or PDF files.

The extension only works when the user manually clicks the export button. Its single purpose is to export the currently open ChatGPT conversation page into a file for saving, printing, sharing, or editing.

## What data the extension accesses

When the user manually triggers an export, the extension may access:

- The content of the currently open ChatGPT conversation page
- Text, formatting, formulas, images, and other visible content needed to generate Word or PDF output
- Basic page information required to run on supported ChatGPT pages

## What the extension does **not** collect

This extension does **not**:

- Sell user data
- Use user data for advertising
- Track browsing activity unrelated to the export feature
- Collect passwords, payment information, health data, or location data
- Upload conversation content to a developer-owned server
- Share exported conversation content with third parties, except where required by the browser to complete the local download requested by the user

## How data is used

The accessed page content is used only to:

- Read the current ChatGPT conversation page selected by the user
- Generate a Word or PDF file requested by the user
- Save the generated file to the user’s device using the browser download function

## Permissions used

The extension may request the following permissions:

- **activeTab**: used to temporarily access the current active ChatGPT tab when the user clicks export
- **downloads**: used to save the generated Word or PDF file to the user’s device
- **Host permissions for ChatGPT pages**: used to run the extension on supported ChatGPT conversation pages

These permissions are used only for the extension’s single purpose.

## Data storage

The extension does not maintain a remote database of user conversations.

Any generated export files are saved locally to the user’s device through the browser’s download process.

If the extension stores user preferences such as language selection or export mode, such settings are stored locally in the browser and are not sold or shared for advertising purposes.

## Remote code

The extension does not load or execute remotely hosted JavaScript or WebAssembly code. Its functional code is packaged with the extension.

## Third-party services

This extension does not use third-party analytics, advertising, or tracking services for its core export functionality.
