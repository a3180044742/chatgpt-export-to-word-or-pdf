# chatgpt export to word/pdf

中文 | [English](#english)

一个简单实用的 Chrome 扩展，用于将 **ChatGPT 对话导出为 Word 或 PDF**。

适合学习笔记、技术文档、科研记录、写作整理、工作归档和内容分享。  
支持 **公式保真 Word 导出**、**快速导出 Word** 和 **中文 / English 界面切换**。

## 功能特点

- 一键导出 ChatGPT 对话为 **PDF**
- 一键导出 ChatGPT 对话为 **PNG**
- 支持 **Word（公式保真）** 导出，适合数学公式、技术内容和科研笔记
- 支持 **Team 绑卡链接生成**
- 支持 **中文 / English** 界面切换
- 保留清晰的对话结构，便于阅读、整理、打印和分享

## Word 导出模式

### Word（公式保真）
适合包含数学公式、技术推导和科研内容的对话。  
导出效果更接近 ChatGPT 页面中的公式显示效果。

### 快速导出 Word
适合普通文本和较长对话。  
导出速度更快，更适合快速保存和归档。

## 适用场景

- 保存 ChatGPT 学习笔记
- 导出带公式的解题过程
- 整理 AI 生成内容
- 归档研究和工作对话
- 分享或打印聊天结果

## 安装方式

### 方式一：从 GitHub 下载
1. 下载本项目的最新版本 ZIP
2. 解压到本地文件夹
3. 打开 Chrome，进入 `chrome://extensions/`
4. 打开右上角 **开发者模式**
5. 点击 **加载已解压的扩展程序**
6. 选择解压后的项目文件夹

## 使用方法

1. 打开 ChatGPT 对话页面
2. 点击扩展图标
3. 选择导出方式：
   - 导出 PDF
   - 导出 Word（公式保真）
   - 快速导出 Word
4. 等待文件生成并保存

## 隐私说明

本扩展仅在你主动点击导出时读取当前 ChatGPT 页面内容，用于在本地生成 Word 或 PDF 文件。  
不上传到开发者服务器，不出售用户数据，不用于广告或跟踪。

# English

A simple and practical Chrome extension for exporting **ChatGPT conversations to Word or PDF**.

It is useful for study notes, technical documents, research logs, writing workflows, work archives, and content sharing.  
Supports **formula-preserving Word export**, **fast Word export**, and a **Chinese / English interface**.

## Features

- Export ChatGPT conversations to **PDF**
- Export ChatGPT conversations to **PNG**
- **Formula-Preserving Word Export** for math, technical, and research content
- **Team checkout panel**
- **Chinese / English** language switch
- Clean conversation structure for easier reading, editing, printing, and sharing

## Word Export Modes

### Word (Formula Preserving)
Best for conversations containing equations, technical content, and research notes.  
Provides a result closer to how formulas appear in ChatGPT.

### Fast Word Export
Best for general text and long conversations.  
Faster and more suitable for quick saving and archiving.

## Use Cases

- Save ChatGPT study notes
- Export conversations with equations
- Organize AI-generated writing
- Archive research and work discussions
- Share or print chat results

## Installation

### Option 1: Download from GitHub
1. Download the latest ZIP release of this project
2. Extract it to a local folder
3. Open Chrome and go to `chrome://extensions/`
4. Enable **Developer mode**
5. Click **Load unpacked**
6. Select the extracted project folder

## How to Use

1. Open a ChatGPT conversation page
2. Click the extension icon
3. Choose an export mode:
   - Export PDF
   - Export Word (Formula Preserving)
   - Fast Export Word
4. Wait for the file to be generated and saved

## Privacy

This extension only reads the current ChatGPT page when you manually choose to export it, in order to generate Word or PDF files locally.  
It does not upload data to any developer server, does not sell user data, and is not used for advertising or tracking.
